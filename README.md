## 📋 Обзор проекта

**OpenSolarUPS** — это умный ИБП (Источник Бесперебойного Питания) с интегрированным MPPT-контроллером для одновременного питания роутера и ONU (оптического модуля). Система отображает официальные графики отключений на цветном сенсорном экране и обеспечивает безопасную работу оборудования в условиях нестабильной сети.

### 🎯 Ключевые возможности

- ☀️ **Прямая зарядка от солнечных панелей** (MPPT CN3795, до 50W)
- 🔋 **LiFePO4 батареи** (12.8V, 4S1P, срок службы 5-7 лет)
- 📊 **Цветной сенсорный дисплей** (2.4"-2.8" ST7789/ILI9341)
- 📈 **Авто-парсинг расписания облэнерго** по Wi-Fi
- ⚡ **Раздельный Hard Reset** роутера и ONU
- ❄️ **Умный подогрев батарей** при T <= +2°C
- 📱 **Мобильное приложение** (Flutter + Firebase)
- 🔒 **Flash Encryption + Secure Boot** для защиты ПО
